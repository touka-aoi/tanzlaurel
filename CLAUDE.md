## プロジェクト概要
<!-- プロジェクトの概要を簡潔に説明してください。-->
WebSocketベースのリアルタイム通信システム（シューティング+AIボット）
**使用言語** Go 1.26 / TypeScript (Vite)
**主要な依存関係** github.com/coder/websocket, go.uber.org/mock

## コマンド
<!-- ワークフローの変更に合わせて更新すること - コマンドは頻繁に変わる -->
### 開発
`make dev`（一括起動）、個別は `make server` / `make client` / `make bot`

### テスト
`make test`（Go）、`make e2e`（Playwright）

### ビルド
`cd client && npm run build`
